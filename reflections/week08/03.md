## Questions

1. What is Software Development Testing?

Testing in software development is almost the same thing as testing any other big project, you make sure each component works as intended and move on to testing the next component.

2. What are the benefits of Testing in software development?

The benefits of testing in software development is you can usually identify the problem easier if you are testing regularly and one component or function at a time.

3. What are some potential drawbacks of Testing in software development?

Testing every little part of you software and getting stuck on one test can lead to losing time to build out the project, many developers aren't in the habit of testing components in isolation because the cost of building the entire system is usually very low.
